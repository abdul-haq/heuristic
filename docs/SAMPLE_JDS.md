## Sample 1 — Full-stack Werkstudent (English, fintech)

**Title:** Working Student Full-Stack Developer (m/f/d)
**Company:** Finanz GmbH
**Location:** Frankfurt am Main, hybrid

We are a Frankfurt-based fintech building APIs for the European banking sector. We're looking for a working student to join our platform team for 15-20 hours per week.

**You will:**
- Build features across our Node.js / TypeScript backend and React frontend
- Design and implement REST APIs backed by PostgreSQL
- Work with Docker containers in our development environment
- Participate in code reviews and bi-weekly sprints

**You bring:**
- Strong knowledge of JavaScript or TypeScript
- Experience with React and a Node.js backend framework
- Familiarity with PostgreSQL or another relational database
- Git, basic Docker
- Working English (German is a plus but not required)

Bonus: experience with NestJS, GraphQL, or AWS.

## Sample 3 — Werkstudent GenAI Engineer (English, startup)

**Title:** Working Student GenAI Developer
**Company:** Helios Labs
**Location:** Darmstadt, remote-friendly

Helios Labs builds AI-powered tools for legal teams. We're hiring a working student (15-20h/week) to help us ship LLM features.

**Responsibilities:**
- Build LLM-powered features in our TypeScript backend
- Implement RAG pipelines with vector databases (we use pgvector)
- Work on prompt engineering and structured output extraction
- Help evaluate LLM output quality

**Must-haves:**
- Strong TypeScript or Python
- Hands-on experience with LLM APIs (OpenAI, Anthropic, or open-source)
- Understanding of embeddings, RAG, and vector search
- Backend experience (Node.js, FastAPI, or similar)

**Nice-to-have:**
- Experience with LangChain or building agents
- Familiarity with prompt evaluation frameworks
- German not required, working English is enough
